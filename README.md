# APOD Search App – Assignment Part 4

## Overview
This project is a small web application that lets users explore NASA’s Astronomy Picture of the Day. By selecting a specific date, the app retrieves the image and its information using the NASA APOD API. Users can also save images they like to a favourites list so they can easily view them again later.

## Features
- Search for the Astronomy Picture of the Day by selecting a date  
- Display the image along with its title, date, and explanation  
- Prevent users from selecting future dates  
- Save favourite images  
- View saved favourites  
- Simple and responsive layout  

## Technologies Used
The project was built using basic web technologies:
- HTML
- CSS
- JavaScript
- NASA APOD API
- Browser Local Storage for saving favourites

## How the App Works
When the user visits the page, they can choose a date using the date input field.  
The app then sends a request to the NASA APOD API and retrieves the image for that day.  

The image, title, and description are displayed on the page. If the user likes the image, they can save it to their favourites. The favourites are stored in the browser using local storage, so they remain available when the user returns to the site.

## Running the Project
To run the project:

1. Download or clone the repository
2. Open the project folder
3. Open the `index.html` file in your browser

No installation or additional setup is required.

## API
This project uses the NASA Astronomy Picture of the Day API:  
https://api.nasa.gov/
